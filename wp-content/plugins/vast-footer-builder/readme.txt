This plugin is an additional plugin for Vast Theme from Detheme
 
== Description ==
 
This Plugin allows you to efficiently uses Vast Theme from Detheme.
The features that provided by this plugins are:
- Video Guide, this feature allows the users to efficiently uses Vast Themes via video guide
- WooCommerce Rating Add-on, this feature allows you to add a rating system on WooCommerce products that works beautifully and aesthetically consistent with the Vast Theme
- Dashboard Support System, this plugin allows you to use a Helpscout help service, this allows you to be able to access and contact our support team without even leaving the WordPress dashboard.
You can find out more about the Helpscout service here: http://helpscout.net
To find out more about their privacy policy read more here: https://www.helpscout.net/company/legal/privacy/
For Helpscout’s GDPR policy: https://www.helpscout.net/company/legal/gdpr/
